# Engenharia de atributos

Este projeto aborda a etapa de Engenharia de Atributos, contendo:
- Limpeza e transformação de dados
- Feature selection e extraction
- Recategorização de variáveis
- Recoding e agrupamento de variáveis
- Entrega de dataset tratado e limpo

A área escolhida foi a área médica, com dados contendo informações sobre pacientes diabéticos e foi baixado do UCI Machine Learning Repository.

http://archive.ics.uci.edu/ml/datasets/diabetes+130-us+hospitals+for+years+1999-2008

Os dados representam 10 anos (1999-2008) de atendimento clínico em 130 hospitais dos EUA e redes de distribuição integradas com mais de 100.000 observações e 50 recursos (variáveis) que representam os registros eletrônicos com resultados de exames dos pacientes e dados sobre cada hospital.
